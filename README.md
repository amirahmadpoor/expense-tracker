# Expense Tracker

یک پروژه تمرینی مدیریت هزینه با React برای یادگیری و تقویت مفاهیم پایه React.

## معرفی پروژه

هدف این پروژه تمرین مفاهیم React و ساخت یک پروژه کوچک اما کاربردی در مسیر یادگیری فرانت‌اند است.
موضوع اصلی پروژه، مدیریت هزینه‌ها و درآمدهای شخصی است.

در حال حاضر پروژه فقط شامل UI است و منطق اصلی (ثبت داده، محاسبات، فیلتر و چارت) هنوز پیاده‌سازی نشده.
این قابلیت‌ها به صورت مرحله‌ای و تدریجی اضافه می‌شوند.

## وضعیت فعلی

- UI اولیه صفحات و کامپوننت‌ها آماده است.
- منطق مدیریت هزینه/درآمد هنوز فعال نشده است.

## قابلیت‌های هدف (در نسخه‌های بعدی)

- اضافه کردن تراکنش جدید
- تعیین دسته‌بندی برای هر تراکنش (مثلا خوراک، حمل‌ونقل، تفریح و ...)
- تعیین نوع تراکنش (هزینه یا درآمد)
- وارد کردن مبلغ تراکنش
- نمایش مجموع درآمد
- نمایش مجموع هزینه
- نمایش موجودی کیف پول (درآمد - هزینه)
- فیلتر کردن تراکنش‌ها براساس دسته‌بندی
- نمایش چارت درآمد و هزینه

## تکنولوژی‌ها

- React
- Vite
- Tailwind CSS

## اجرای پروژه

پیش‌نیاز: Node.js نسخه 18 یا بالاتر

```bash
npm install
npm run dev
```

پس از اجرا، پروژه در آدرس لوکال نمایش داده می‌شود (معمولا):

```text
http://localhost:5173
```

## برنامه توسعه

1. تکمیل فرم افزودن تراکنش
2. ذخیره‌سازی لیست تراکنش‌ها در state
3. محاسبه مجموع‌ها و موجودی کیف پول
4. پیاده‌سازی فیلتر براساس دسته‌بندی
5. اضافه کردن چارت برای نمایش درآمد و هزینه
